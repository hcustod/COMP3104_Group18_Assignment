
# COMP3104 DevOps Course

The COMP3104 DevOps course introduces students to modern software 
development practices that combine development and operations. 
It covers topics such as version control with Git and GitHub, 
Continuous Integration (CI), Continuous Deployment (CD),
and automation workflows using tools like GitHub Actions and Travis CI. 
Through hands-on projects, students learn to collaborate effectively
and maintain high-quality, reliable code in shared repositories.
